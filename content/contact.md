# Contact and Evidence Submission

This page defines the intended public contact rule for punnaraj.com.

If a visitor needs ordinary public information, the website or chatbot may answer normally.

If a visitor reports a problem, harm, rights concern, factual concern, or reputational impact, the system must route the visitor to a real responsible person.

This contact path must not be another chatbot loop.

## Public contact fields

Fill these before production publication:

- Responsible person: [name]
- Email: [public contact email]
- Phone: [optional public phone]
- Evidence submission channel: [email or form]
- Postal or document address: [optional public correspondence address]

## Evidence requested

When reporting a problem, include as much of the following as possible:

- page URL or chatbot session reference
- date and time
- screenshot or copied text
- description of the impact
- requested correction or response
- contact information for follow-up

Responsibility means receiving the report, checking evidence, correcting or clarifying where appropriate, and recording the action taken.
